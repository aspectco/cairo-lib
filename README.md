## Installation

To install with pip

```sh
pip install cairo-lib
```

To install with [**Hardhat**](https://github.com/nomiclabs/hardhat) or [**Truffle**](https://github.com/trufflesuite/truffle):

```sh
npm install cairo-lib
```
